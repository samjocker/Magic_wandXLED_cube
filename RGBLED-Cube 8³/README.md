# CubeDrive V0.1 -pwm.test
## **此版本為digital模擬LED的pwm測試代碼**
### 版本紀錄:
  利用二進位減少LED每禎的寫入次數
### 預計效果:
  由原255次刷新改進為8次刷新，減少每禎消耗時間以提升整體刷新率。
